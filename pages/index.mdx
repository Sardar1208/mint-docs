
## Getting started

Just wrap your component with **AnimatedWrapper** to enable micro interactions on it. Simple!

```js
import { AnimatedWrapper } from 'react-native-micro-interactions';

<AnimatedWrapper>
    // Your component
</AnimatedWrapper>
```
