# Pop In

This animation is generally used when the component is first initialised. Check out [Group Animations](/group_animations) to find a better use case for this animation

<Image src="/pop_in.gif" width={300} height={100} style={{marginTop: "20px"}}/>

### Animation Options

Customize your animation using these options. 

Check [Customization](/Usage#customization) for more info!

| **Options**        | **Description**                                                                              | **Default**                       |
|---------------------|----------------------------------------------------------------------------------------------|-------------------------------------|
| withBounce  | enable/disable bounce effect in pop in animation                                            | true                                |
| duration| How long should it take to slide in.                                                       | 250 (ms)           |


### Implementation

```js
import { AnimatedWrapper } from 'react-native-micro-interactions';

return(
    <AnimatedWrapper animationTrigger='init' animationType='pop_in'>
        <Text>Hello World!!</Text>
    </AnimatedWrapper>
)
```
